Website Review

Submission leads directly to the reviewed page

There are no spaces or upper-cases in the file/folder names, including scripts, images, etc.

Design

The page appears to have sufficient contrast and font sizing, making it easy to read.

There is a consistent color scheme that matches the logo of the Continental Inn, indicating the use of a standard CSS file or a theme.

The principles of CRAP (Contrast, Repetition, Alignment, Proximity) seem to be adhered to:

  • There is good contrast between the text and the background.
  • The design seems to repeat certain elements, like font and color schemes, to create a unified look.
  • Alignment is consistent across different elements of the page.
  • Proximity is maintained in the layout to group related items together, such as the navigation links at the top of the page.

Page Composition

The page includes:

  • A header at the top of the page which includes navigation links.
  • A main content area that displays an image of the hotel lobby.
  • A footer at the bottom of the page, which includes the address and phone number of the Continental Inn.
  • A navigation menu in the header, serving as the main menu between the header and the main content.

Header Details

The header includes the site/brand name "Continental Inn" with text that likely serves as an h1 element, as it is prominent and probably the title of the page.
